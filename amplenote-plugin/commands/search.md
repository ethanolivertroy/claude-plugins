Search for notes in your Amplenote account.

Usage:
  /amplenote:search <query>

Examples:
  /amplenote:search "meeting notes"
  /amplenote:search project
  /amplenote:search "Q2 planning"

Returns matching notes with:
- Note title
- UUID (for use with other commands)
- Tags

Use the returned UUID with `/amplenote:create-task` to add tasks to notes.
