Create a new note in Amplenote.

Usage:
  /amplenote:create-note <title> [content]

Examples:
  /amplenote:create-note "Meeting Notes"
  /amplenote:create-note "Project Ideas" "Initial brainstorm for Q2 projects"
  /amplenote:create-note "Daily Journal"

Options:
  <title>    - Note title (required)
  [content]  - Initial note content (optional)

The note will be created in your Amplenote account. Returns the note UUID for future reference.
